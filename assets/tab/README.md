# TabBar 图标文件

本目录需要放置 6 个 PNG 图标文件，用于底部 TabBar 导航。

## 文件清单

| 文件名 | 用途 | 尺寸 | 说明 |
|--------|------|------|------|
| calendar.png | 日历 Tab 默认态 | 81 x 81 px | 灰色线条图标 |
| calendar-active.png | 日历 Tab 选中态 | 81 x 81 px | 橙红色 (#E85D3A) 填充图标 |
| solar.png | 节气 Tab 默认态 | 81 x 81 px | 灰色线条图标 |
| solar-active.png | 节气 Tab 选中态 | 81 x 81 px | 橙红色 (#E85D3A) 填充图标 |
| holiday.png | 节假日 Tab 默认态 | 81 x 81 px | 灰色线条图标 |
| holiday-active.png | 节假日 Tab 选中态 | 81 x 81 px | 橙红色 (#E85D3A) 填充图标 |

## 设计规范

- 格式: PNG（支持透明背景）
- 尺寸: 81 x 81 px（微信推荐尺寸）
- 默认态颜色: #999999（灰色）
- 选中态颜色: #E85D3A（主题橙红色）
- 风格: 简洁线条风格，与微信原生图标一致

## 图标内容建议

- **calendar**: 日历网格图标（带日期数字的方格）
- **solar**: 太阳/叶子/自然元素（代表节气）
- **holiday**: 旗帜/礼物/星星（代表节假日）
